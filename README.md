# MovieHub

## TMDb API Setup

This project uses The Movie Database (TMDb).

### Setup
1. Create a TMDb account
2. Generate an API Read Access Token
3. Copy `Secrets.example.xcconfig` to `Secrets.xcconfig`
4. Put your token into `TMDB_READ_TOKEN`
5. Build & run

`Secrets.xcconfig` is ignored by git and must not be committed.
